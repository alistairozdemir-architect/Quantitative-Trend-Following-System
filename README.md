# Quantitative-Trend-Following-System
Systematic trend-following system built with Python and backtested in QuantConnect. SPY + QQQ portfolio, benchmark comparison, drawdown analysis, and transaction-cost-aware backtests.

Long-only trend-following portfolio research for SPY and QQQ, developed in Python and QuantConnect.

Period:
2008–2026

Benchmark:
SPY + QQQ Equal Weight Buy & Hold

Transaction Costs:
Commissions and slippage included.
